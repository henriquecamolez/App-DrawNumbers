## ✨ Sobre o Projeto

O **App DrawNumbers** é um aplicativo Android desenvolvido com foco em aprendizado e prática com Kotlin.

Ele permite gerar números aleatórios dentro de um intervalo definido pelo usuário — ideal para:

- 🎯 Sorteios
- 🎮 Jogos
- 📚 Estudos de lógica
- 🎲 Decisões aleatórias rápidas

Projeto criado com foco em:
- Organização de código
- Estrutura Android moderna
- Boas práticas de desenvolvimento

---

## 🚀 Funcionalidades

- 🔢 Definição de intervalo mínimo e máximo
- 🎲 Geração de número aleatório
- 📱 Interface simples e funcional
- ⚡ Execução rápida e leve

---

## 🧠 Stack utilizada

- **Android SDK**
